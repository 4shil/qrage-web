# QRAGE Web: A QR Code Generator with Neo-Brutalism Design

QRAGE Web is an advanced QR code generation application tailored to meet modern web design expectations with a Neo-Brutalism aesthetic. Its focus is on providing the best performance while ensuring privacy and offline functionality.

## Features
1. Generate QR codes for:
   - URLs
   - Wi-Fi Connections
   - Social Media Profiles (Instagram, X, LinkedIn, WhatsApp, Telegram, GitHub, YouTube)

2. **Customization Options:**
   - Foreground and Background color options
   - Quiet zone adjustments
   - Center logo inclusion (restrained to 20% size)
   - Thick border frames

   _restrictions:_ No transparency, gradients, or distortions allowed for privacy & quick scannable purposes
